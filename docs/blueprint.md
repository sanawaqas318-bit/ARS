# **App Name**: Arham's Portfolio

## Core Features:

- Homepage Showcase: Display a professional profile picture and introductory headline to capture user attention immediately.
- About Section: Provide a concise background, including education and work experience at 'deveryx.dev'.
- Skills Visualization: Visually represent key skills in frontend development, such as responsive layouts and UI implementation.
- Service Offering: Clearly list services like Website Development and UI/Frontend implementation with descriptions.
- Interactive Project Gallery: Showcase completed projects, like restaurant or home decor websites, with descriptions and visual mockups. Hover effects highlight frontend skills.
- Contact Form and Information: Include a contact form and display contact details: email (arham.waqas@gmail.com) and phone (03324800263).

## Style Guidelines:

- Primary color: Bright yellow (#FFDA63) to add a youthful energy, fitting for a tech-savvy developer.
- Background color: Very light yellow (#FAF8F0) to give spaciousness to the white-dominated layout.
- Accent color: Soft orange (#FFB347) for call-to-action buttons and important interactive highlights. It is similar in feeling, but contrasted in saturation, compared to the primary yellow.
- Font pairing: 'Poppins' (geometric sans-serif) for headlines and 'PT Sans' (humanist sans-serif) for body text, giving a contemporary, readable style.
- Use minimalistic line icons to represent skills, services, and contact information, maintaining a modern feel.
- Employ a clean, minimal layout with ample whitespace to ensure clarity and readability across all sections.
- Incorporate smooth transitions and subtle micro-interactions to showcase frontend skills without overwhelming the user.
